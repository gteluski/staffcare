# Project Memory

## Core
Plataforma Pastoral Metodista — pastoral organizer for Methodist pastors in Brazil.
UI language: Brazilian Portuguese only. Technical docs in English.
Stack: React + Vite + TypeScript + Lovable Cloud (Supabase). Mobile-first.
Palette: warm pastoral tones. Helvetica Neue primary, JetBrains Mono secondary only.
Primary: deep navy-teal 200 35% 22%. Colorful icon accent palette (10 semantic colors).
Theological lens: Arminian-Wesleyan / Methodist. Never present opinion as fact.
Users have low digital literacy — maximum simplicity and accessibility.
Gendered greeting: Pastor/Pastora based on profile gender.
Cânones 2023 is the highest-authority source.
